Predicting Finger Flexion Motion

An algorithm to predict finger flexion motion from raw ECoG data. The algorithm pre-processes input data, extracts six features (average time domain signal voltage and average frequency domain voltage of five different frequency bands), makes predictions using a linear decoder, and post-processes the predictions. This algorithm achieved a maximum correlation of 0.5072 when compared to unseen testing data.

An Exploration of Image Processing

A Matlab GUI exploring various image processing techniques in the spatial and frequency domains. Spatial domain processing techniques use convolution with various kernels to produce effects like blurring, shapening, edge detection, and embossing. Frequency domain processing techniques use the fast Fourier Transform to analyze various properties of image's magnitude and phase components.
